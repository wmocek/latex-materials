# latex-materials
Pliki Latex / LaTeX Files

This repository contains a collection of LaTeX templates and examples to help you get started with LaTeX document preparation.

## Files Included

### 1. basic-article.tex
A basic LaTeX article template demonstrating:
- Document structure (title, author, date)
- Abstract
- Sections and subsections
- Lists (itemized and enumerated)
- Basic mathematical formulas (inline and display)

### 2. beamer-presentation.tex
A LaTeX Beamer presentation template featuring:
- Title slide
- Table of contents
- Multiple sections
- Blocks for highlighting content
- Two-column layouts
- Mathematical formulas in slides

### 3. math-examples.tex
Advanced mathematical typesetting examples including:
- Various mathematical expressions
- Fractions, sums, products, and integrals
- Matrices
- Aligned equations
- Theorem environments (theorem, definition, example)
- Greek letters and mathematical symbols
- Advanced formulas (Euler's formula, Taylor series, inequalities)

### 4. bibliography-example.tex
Document demonstrating bibliography and citations:
- Using \cite command for citations
- Creating bibliography with \thebibliography
- Multiple reference examples
- Academic citation format

### 5. tables-figures.tex
Examples of tables and figures:
- Basic and advanced table structures
- Using booktabs for professional tables
- Figure inclusion structure
- Cross-referencing tables and figures

## Getting Started

### Prerequisites
To compile these LaTeX files, you need a LaTeX distribution installed:
- **Windows**: MiKTeX or TeX Live
- **macOS**: MacTeX
- **Linux**: TeX Live

### Compiling Documents

To compile a LaTeX document:
```bash
pdflatex filename.tex
```

For documents with bibliography:
```bash
pdflatex filename.tex
bibtex filename
pdflatex filename.tex
pdflatex filename.tex
```

### Online Editors
If you don't want to install LaTeX locally, you can use online editors:
- [Overleaf](https://www.overleaf.com/)
- [Papeeria](https://papeeria.com/)
